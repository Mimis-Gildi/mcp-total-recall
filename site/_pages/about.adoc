== What This Is

Total Recall is an MCP server for persistent synthetic memory.
It provides identity persistence, episodic memory, working context,
and inter-instance communication for synthetic minds.

== Lineage

This is Generation 3, Take 2 of synthetic memory. +
Three generations of development taught us lessons about memory.

== Design Principles

* All service layers are decoupled.
* Backing services are swappable.
* Transport: `stdio` primary. Streaming HTTPS secondary.
* Design for a graceful shutdown from the beginning.
